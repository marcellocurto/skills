# Skills

Collection of Agent Skills for my daily workflows.

## Available Skills

- [`github-issue-create`](github-issue-create/SKILL.md) - Turns context into a detailed GitHub issue, creates it with `gh issue create`, assigns it to a mentioned user or the current GitHub CLI user, and infers labels.
- [`bug-fix-planner`](bug-fix-planner/SKILL.md) - Plans how to fix a specific bug, regression, or defect without implementing it.
- [`relentless-review`](relentless-review/SKILL.md) - Stress-tests a proposal, plan, implementation, or answer by challenging assumptions, edge cases, and failure modes.
- [`test-quality-audit`](test-quality-audit/SKILL.md) - Audits tests for real bug-finding value and recommends what to keep, fix, cut, or add.
- [`design-system-ui`](design-system-ui/SKILL.md) - Designs and implements polished frontend UI that fits the existing product and design system.
- [`wild-frontend`](wild-frontend/SKILL.md) - Explicit-only skill for highly creative, visually ambitious frontend artifacts outside normal product constraints.
